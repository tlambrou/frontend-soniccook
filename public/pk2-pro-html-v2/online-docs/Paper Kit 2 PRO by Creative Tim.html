Short Description

Paper Kit 2 PRO is a beautiful resource built for Bootstrap. We have redesigned all its components with a pale colour scheme and paper-like transitions. We are proud of this new Bootstrap skin and the possibilities for customisation.

Using it is very simple and it will enable you to refresh you classic looking application. To get the desired effect you will also need to integrate JS plugins that take a little bit more effort. In the end the result will be worth it.

Getting Started


The Paper Kit is built on top of Bootstrap 4, so you can safely use it on your existing or new Bootstrap project. No line of code from Bootstrap 3 was changed, so you don't have to worry about undesired effects on your work.

We provide all the necessary CSS resources. To get going, just include 'css/pk-paper.css' in your HTML template. Your project will get the new look.

To jump start your project, you can use our start-up template where all the files are already included and ready to use. If you do however, want to start from scratch, you can see the file structure below. The core JavaScript is contained in 'js/paper-kit.js'. For the other files, you should add them if you use the specific element inside your page.

Changing Colors (SASS)

You can change the default colors via variables from SASS:

  1. You can find the colors in assets/scss/paper-kit/_variables.scss starting with line 72 where is the primary color set: $brand-primary: $primary-color !default;.
  2. Add the SASS folder from assets/scss/ to a new project inside Koala Compiler and find the file paper-kit.scss, it will be the one in Green.
  3. Right click on that file and set the output path, it must be in assets/css/ so anything that you compile will overwrite the original paper-kit.css
  4. Press on compile and everything will be done automatically.
